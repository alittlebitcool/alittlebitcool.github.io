三分法

​ 适用于在单峰函数（即先递增后递减，或先递减后递增的函数）中寻找极值（最大值或最小值）的算法。如果函数有多个极值点，三分法可能无法正确找到全局极值。它的思想是通过将区间分成三部分。每次将区间 [left, right] 分成三部分：mid1 = left + (right - left) / 3 和 mid2 = right - (right - left) / 3。 通过比较 f(mid1) 和 f(mid2)，缩小搜索范围。通过比较 f(mid1) 和 f(mid2)，缩小搜索范围，最终找到极值点。

回溯算法

​ 回溯算法是一种通过递归尝试所有可能的解，并在不满足条件时回退（撤销选择）的算法。它常用于解决组合、排列、子集等问题。

​ 关键点：路径（Path）： 表示当前已经做出的选择。 通常用 List 或 StringBuilder 等数据结构存储。 选择列表（Choices）： 表示当前可以做的选择。 在排列问题中，选择列表是所有未使用的元素；在组合问题中，选择列表是剩余的元素。 结束条件（Base Case）： 当路径满足某种条件时，将其加入结果集。 例如，路径长度等于数组长度（排列问题），或者路径和等于目标值（组合总和问题）。 剪枝（Pruning）： 在遍历选择列表时，跳过无效的选择，减少递归次数。 例如，跳过已经使用过的元素，或者跳过不符合条件的选择。 做选择与撤销选择： 在递归调用前，将选择加入路径。 在递归调用后，将选择从路径中移除，恢复状态。

​ 每一次独立的决策只对应「选择」和「不选」两种情况，适用于每个元素只有两种选择的情况，例如子集问题、组合问题等。

​ 每一次独立的决策对应多种选择这种形式适用于每个决策有多种选择的情况，例如排列问题、组合总和问题等。
